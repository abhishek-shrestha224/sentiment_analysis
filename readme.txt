1. Unzip the zip file and extract the folder called "Tweet Sentiment Analysis"

2. Here you will find multiple files. 

3. The first file is the dataset named "Tweets.csv".

4. The second file is the jupyter notebook file named "SentimentAnalysis.ipynb"

5. The third file is a video file that gives a brief overview of the prototype model named "SentimentAnalysisVideo.mkv".

6. The final file is the report of prototype model named "Report.pdf".
